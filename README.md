<# Finance Dashboard

A modern finance dashboard built using React and Vite.

## 🚀 Features
- Track income and expenses
- Interactive charts and insights
- Clean and responsive UI
- Transaction management

## 🛠️ Tech Stack
- React
- Vite
- Tailwind CSS
- Chart.js

## 📂 Project Structure
- src/components → UI components
- src/pages → Pages like Dashboard, Insights
- src/context → Global state management
- src/data → Mock data

## ▶️ Run Locally
```bash
npm install
npm run dev
